GESTIÓN DE CARRERAS v1.0

Presentación:

La aplicación "Gestión de Carreras" permite de una forma sencilla e intuitiva manejar un simulador de carreras deportivas

La principal característica de la aplicación es la posibilidad de manejar un listado de corredores dados de alta y un listado de carreras para simular la disputa de las mismas


Listado de funcionalidades destacadas:

  • Crear, modificar, eliminar o importar corredores.
  • Crear, modificar, eliminar carreras.
  • Manejar los corredores participantes en una carrera.
  • Simular la disputa de una carrera.
  • Simular la clasificación de una carrera durante su disputa.
  • Generación de informes tras finalizar la simulación.
  • Opción para personalizar el estilo visual de las ventanas.
  • Guardado automático de datos, personalizable.

Ficheros de salida de datos:

Los ficheros de almacenamiento de datos se situán en el directorio raiz de la aplicación y se cargan cada vez que la aplicación es iniciada, y se sobreescriben despues de cada guardado de datos

  • Datos de corredores.: ListadoDeCorredores.dat
  • Datos de carreras.: ListadoDeCarreras.dat
